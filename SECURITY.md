# 🛡️ Security Policy

## Supported Versions

Currently, only the latest release of the Freedom & Peace Party app is supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| v0.1.x  | :white_check_mark: |
| < v0.1  | :x:                |

## Reporting a Vulnerability

Because this app handles anonymous voting and ZK-proof cryptography, **security is our highest priority**.

If you discover a potential security vulnerability in this project, **do not open a public issue.** 

Please email us directly at `security@freedom-peace-party.org` (or the direct maintainer contact). We will:
1. Acknowledge your report within 48 hours.
2. Provide a timeline for the fix.
3. Notify you when the patch is deployed and publicly credit you (if desired) after the vulnerability is safely patched.

Thank you for helping keep global democratic participation secure.
